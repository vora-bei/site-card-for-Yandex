Вопрос №2

Опишите своими словами, что происходит в браузерах при клике на ссылку yandex.ru (полнота ответа — на ваше усмотрение).

При клике на ссылку возникает событие click, которое передается всем обработчикам на странице, порядок вызовов которых определен моделью всплытия, что означает, что после того как они возникли для целевого элемента, они по очереди возникают для каждого предка в DOM-дереве, вплоть до объекта document включительно. Всплытие может быть прервано и тогда событие по умолчанию переход по url не происходит. Потом браузер обрабатывает событие click которому передается элемент на котором возникло событие (тег а),в соответствии с аттрибутами ссылки(в новой вкладке или нет,например) браузер открывает новую страницу по нужному урл.

#!/usr/bin/env perl

use strict;
use Getopt::Long;
use Pod::Usage;

my $help;
my $message = '';
my $verbose;

@ARGV and GetOptions(
    "h" => \$help,
    "m:s" => \$message,
	"v" => \$verbose
    # TODO: add '-v' option for verbose mode
) or pod2usage(1);
pod2usage(-verbose => 2, -exitval => 2) if $help;

my $count = 0;

foreach $arg(@ARGV) {
    # TODO: add '-v' option for verbose mode
    # and print each argument
	print($arg . "\n") if $verbose;
    $count++;
}

if($message ne "") {
    print($message . "\n");
}
